📌 Project Overview

The Digital Clock & Countdown Timer is a simple web application built using HTML, CSS, and JavaScript. It displays the current time in HH:MM:SS format and includes a countdown timer that updates every second until New Year's Day. This project demonstrates how to use JavaScript's Date() object, setInterval(), and DOM manipulation to create real-time applications.

🚀 Features
🕒 Live Digital Clock (HH:MM:SS)
⏳ Countdown Timer to New Year
🔄 Real-time updates every second
📱 Responsive design for desktop and mobile
🎨 Attractive modern UI with gradient background
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
